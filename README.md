# Muhammad Niaz Ali — Portfolio

A modern, ultra-advanced portfolio website built with **Next.js 14**, **TypeScript**, **Tailwind CSS**, **Three.js**, and **Framer Motion**.

## 🚀 Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **3D Graphics**: Three.js + React Three Fiber + Drei
- **Icons**: Lucide React
- **Type Animation**: React Type Animation
- **Notifications**: React Hot Toast
- **Counter**: React CountUp

## ✨ Features

- 🌑 Black background with white/green color scheme
- 🎯 Custom animated cursor (desktop)
- 🌐 3D animated hero scene (Three.js)
- ✍️ Typing animation with roles
- 📊 Animated skill bars & stats
- 🗂️ Filterable projects grid
- 📬 Contact form with validation
- 📱 Fully responsive & mobile-first
- ⚡ Framer Motion page animations
- 🔤 Beautiful fonts: Bebas Neue, Space Grotesk, DM Sans, JetBrains Mono
- 🎞️ Infinite scrolling tech marquee
- 🌟 Noise texture, grid background, glow effects

## 🛠️ Setup & Installation

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# 1. Navigate to project folder
cd muhammad-niaz-ali-portfolio

# 2. Install dependencies
npm install

# 3. Run development server
npm run dev

# 4. Open in browser
# http://localhost:3000
```

### Build for Production

```bash
npm run build
npm start
```

## 📦 Deployment

Deploy easily to **Vercel**:

1. Push code to GitHub
2. Import repo on [vercel.com](https://vercel.com)
3. Click Deploy — done!

Or use **Netlify**, **Railway**, or any Node.js hosting.

## 🎨 Customization

### Personal Info
Update your details in:
- `app/layout.tsx` — SEO metadata
- `components/sections/HeroSection.tsx` — name, roles, description
- `app/about/page.tsx` — bio, timeline
- `app/projects/page.tsx` — your projects
- `app/contact/page.tsx` — contact info, socials
- `components/Footer.tsx` — social links

### Colors
Edit `tailwind.config.ts` and `app/globals.css`:
```css
--accent: #00ff88;   /* Green accent */
--accent2: #0088ff;  /* Blue accent */
```
